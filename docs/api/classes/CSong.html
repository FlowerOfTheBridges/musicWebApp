show()

show(integer  $id)

La funzione show permette la visualizzazione della canzone da parte di un utente. Se l'utente può effettivamente visualizzarla, sarà possibile riprodurla, altrimenti verrà mostrato un messaggio d'errore. In caso la canzone sia visualizzata dall'artista stesso o da un moderatore, sarà possibile visualizzare funzionalità come la modifica o la rimozione.

Parameters

integer $id
