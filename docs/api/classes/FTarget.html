Foundation\FTarget.php
Package
Foundation La classe FTarget contiene stringhe da utilizzare come parametro per FPersistantManager per specificare, in funzionalita come load o exists, i tipi di query che si vogliono utilizzare per una data classe. In particolare: EUser supporta: - EXISTS_NICKNAME - EXISTS_MAIL - EXISTS_USER - LOAD_FOLLOWERS - LOAD_FOLLOWING - LOAD_SUPPORTERS - LOAD_SUPPORTING ESong supporta: - EXISTS_SONG - LOAD_MUSICIAN_SONG ETarget supporta: - LOAD_MOD_REPORTS EFollower supporta necessariamente: - EXISTS_FOLLOWER ESupporter supporta necessariamente: - EXISTS_SUPPORTER
