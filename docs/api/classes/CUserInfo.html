\CUserInfo

La classe CUserInfo implementa la funzionalità 'Gestione Profilo': le sue funzioni infatti presentano/ricevono una form in cui l'utente inserirà informazioni su di se, come : - Nome e Cognome (se non è un Musician) - Info - Data di Nascita - Luogo di Nascita - Immagine del profilo

editInfo()

A seconda del tipo di metodo richiesto dal client, verranno attivate funzioni diverse.
