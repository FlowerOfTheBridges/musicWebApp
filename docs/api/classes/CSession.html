\CSession

La classe CSession regola la sessione dell'utente nella navigazione dell'applicazione.

Le sue funzionalità permettono di iniziare, terminare e riprendere una sessione di un particolare utente, costruendo/ricostruendo i suoi parametri principali (quali nome, tipologia...)

getUserFromSession()

getUserFromSession() : \EUser

Restituisce l'utente della sessione corrispondente alla connessione che ha richiamato il metodo. Se la sessione è effettivamente attiva, restituirà l'utente corrispondente, altrimenti restituirà un semplice utente guest.
